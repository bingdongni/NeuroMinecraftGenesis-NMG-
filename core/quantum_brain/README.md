# 量子-类脑计算融合系统 (Quantum-Brain Computing Fusion System)

## 系统概述

本系统实现了一个革命性的类脑计算和量子计算深度融合平台，集成了多项前沿技术：

### 核心组件

1. **10万神经元脉冲神经网络实时模拟**
   - 支持多层次神经网络架构
   - 实时STDP（脉冲时序依赖可塑性）学习
   - 可塑性建模和神经连接动态调整

2. **量子决策电路和叠加态探索**
   - 量子态叠加和纠缠
   - 量子门操作（Hadamard, Pauli-X/Y/Z, RX/RY/RZ）
   - 量子决策电路和概率测量
   - 叠加态探索算法

3. **神经符号混合推理架构**
   - 神经网络特征提取
   - 符号概念学习和推理
   - 知识库构建和概念层次
   - 神经-符号结果融合

4. **STDP学习规则和可塑性建模**
   - 实现突触可塑性机制
   - LTP/LTD学习规则
   - 膜电位动态模拟
   - 脉冲传播和连接管理

5. **量子纠缠和相干性应用**
   - 量子相干性计算
   - 纠缠态生成和操作
   - 量子-经典混合决策
   - 叠加态利用和探索

## 配置参数

### 系统配置
```python
fusion_system = QuantumBrainFusion(
    n_neurons=100000,    # 神经元数量
    n_qubits=6          # 量子比特数量
)
```

### 融合参数
- `quantum_classical_weight`: 量子-经典权重 (默认0.5)
- `fusion_threshold`: 融合决策阈值 (默认0.7)

### 量子电路参数
- `decision_threshold`: 量子决策阈值 (默认0.5)

### STDP学习参数
- `A_plus`: 突触增强幅度 (默认0.1)
- `A_minus`: 突触抑制幅度 (默认0.05)
- `tau_plus`: 增强时间常数 (默认20.0)
- `tau_minus`: 抑制时间常数 (默认20.0)
